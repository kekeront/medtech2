Загрузка прайсов — тест парсинга

Upload a .zip bundle (or a single .pdf/.docx/.doc/.xlsx/.xls) to POST /upload. The server unzips it, routes each document to its parser (office → spreadsheet/word; PDF → OCR or text), ingests it, and returns a per-file report.

Leave base URL empty when this page is opened via the API (e.g. http://localhost:8000/ui/upload-test.html). Set it to http://localhost:8000 only if you opened the file directly.
